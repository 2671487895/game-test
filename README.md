# 🐍 贪吃蛇

一个纯 HTML/CSS/JS 实现的经典贪吃蛇小游戏，无需任何依赖，浏览器打开即可畅玩。

## 🎮 玩法

- **方向键 ↑↓←→** 或 **WASD** 控制蛇的移动
- **空格键** 暂停 / 继续
- **R 键** 游戏结束后重新开始
- **手机端** 支持触屏滑动 + 底部方向按钮

## ✨ 特性

- 🐍 蛇身渐变色 + 蛇头带眼睛动画，朝向跟随移动方向
- 🍎 红色食物带脉冲呼吸动画和发光效果
- 💥 吃到食物时触发粒子爆炸特效
- ⚡ 分数越高蛇移动越快，难度递增
- 🏆 最高分自动保存到本地，刷新页面不丢失
- 📱 响应式布局，完美适配手机和电脑
- 🎨 深色主题 + 网格背景

## 🚀 运行

直接用浏览器打开 `index.html` 即可：

```bash
# 克隆仓库
git clone https://github.com/2671487895/game-test.git
cd game-test

# 用浏览器打开
open index.html   # macOS
start index.html  # Windows
```

或者直接双击 `index.html` 文件。
